- 总是保留第一个出现的重复节点
  - if重复，删掉
  - else 转到下一节点
``` java
class Solution {
    public ListNode deleteDuplicates(ListNode head) {
        if(head == null) return head;
        ListNode dummy = head;
        while(head.next != null){
            if(head.next.val == head.val){
                head.next = head.next.next;
            }else{
                head = head.next;
            }
        }
        return dummy;
    }
}
```
